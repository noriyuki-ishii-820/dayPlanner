
## Work Day Scheduler

This is a simple calendar application that allows the user to save events for each hour of the day. 
This app will run in the browser and feature dynamically updated HTML and CSS powered by jQuery.

# Tech Used

- HTML
- CSS
- JavaScript
- Moment.js 
- jQuery
- Bootstrap
